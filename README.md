### 나에게 맞는 개발 포지션은?(My devposition)
한국외대(글로벌) 멋쟁이 사자처럼 12기 아기사자 모집 홍보용 사이드 프로젝트입니다.
간단한 심리 테스트 형식으로, 주어진 질문에 대하여 어떤 답변을 선택하느냐에 따라 다양한 결과가 나오게 됩니다.

**테스트 시작하기 > https://likelion12th-mydevposition.web.app**
